# React Task Status

# Setup 
* git clone https://github.com/ashutoshji/task-todo-app.git
* cd task-todo-app
* npm install
* npm run start

## Features
Task features
* Single board
* Re-orderable lists and tasks
* Table view of tasks

Task functionality
* Title, description, and type
* Colors to indicate types
* Editable tasks
* Create a task

## Tools Used
* React
* Material-ui
* React-Beautiful-dnd
* Styled components
